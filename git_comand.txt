git init: Инициализирует новый репозиторий Git.
git clone <URL>: Клонирует удаленный репозиторий на ваш компьютер.
git status: Показывает состояние файлов в вашем рабочем каталоге и индексе.
git add <файлы>: Добавляет изменения в индекс (staging area) для последующего коммита.
git commit -m "сообщение": Фиксирует изменения в репозитории с определенным сообщением коммита.
git push origin <ветка>: Отправляет изменения из локального репозитория в удаленный репозиторий.
git pull origin <ветка>: Получает изменения из удаленного репозитория и объединяет их с текущей локальной веткой.
git branch: Показывает список всех веток в вашем репозитории.
git checkout <ветка>: Переключается на другую ветку.
git merge <ветка>: Объединяет указанную ветку в текущую ветку.
git log: Показывает историю коммитов.
git diff: Показывает разницу между коммитами, рабочим каталогом и индексом.
git remote -v: Показывает список удаленных репозиториев и их URL.
git rm <файлы>: Удаляет файлы из индекса и из файловой системы.
git reset <файлы>: Снимает файлы с индекса, но сохраняет изменения в рабочем каталоге.
git stash: Сохраняет текущие изменения так, чтобы вы могли их применить позже.
git tag <имя_тега>: Добавляет тег к последнему коммиту.
git fetch: Получает изменения из удаленного репозитория без их объединения.
git checkout -- <файлы>: Восстанавливает файлы из индекса или последнего коммита.
git revert <коммит>: Отменяет указанный коммит, создавая новый коммит с противоположными изменениями.
